simple python randomize number using shell input

How to use:
- MacOS 
  - Ensure the permission to execute: chmod +x randomize.py
  - In a shell, type:
    $python randomize.py 
    OR
    $./randomize.py
  - input the count number you want, then ENTER
  
- Linux
  - Ensure the permission to execute: chmod +x randomize.py
  - In a shell, type:
    $python randomize.py 
    OR
    $./randomize.py
  - input the count number you want, then ENTER
